My Frontend Library

This is a frontend micro-library focusing on data reactivity and HTML data binding.

$NAME gives a small subset of the functionality frameworks like React, Angular, and Vue provide while also being less efficient and more infinite-loop prone.

It also doesn't really have documentation other than this page. I wouldn't recommend using it.

4KB min + gzip.

Supporting:
  • Reactive Data
  • Computed Properties
  • Property Watchers
  • Two-Way Input Binding
  • Declarative, reactive HTML list binding
